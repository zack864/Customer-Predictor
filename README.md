# Customer Predictor [![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://https://customerpredict.streamlit.app/)
https://customerpredict.streamlit.app/- Wake the app if it's sleeping
This a customer purchase predictior web app. It is deployed on Streamlit. The dataset used to train this model is taken from kaggle. The dataset is primarly trained on RandomForests on metrics such as age, gender, income etc. The webapp takes some inputs from the user and gives an output wether the purchase probability is high or low. I hope that you enjoy the app.
